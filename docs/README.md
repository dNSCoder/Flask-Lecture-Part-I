# Flask Lecture Website

โฟลเดอร์ `docs/` คือเว็บไซต์ที่พร้อมใช้กับ **GitHub Pages** (เลือก deploy จาก `/docs`)

- `docs/index.html` = หน้าหลักรวมลิงก์
- `docs/chapters/chapXX/*-doc.html` = เอกสารประกอบ (อ่านยาว)
- `docs/chapters/chapXX/*-present.html` = สไลด์ (กด ← → เลื่อน)

คีย์ลัดในสไลด์:
- ← / → : เปลี่ยนสไลด์
- Home / End : ไปต้น/ท้าย
- F : Fullscreen
- ? : ดูคีย์ลัดทั้งหมด
